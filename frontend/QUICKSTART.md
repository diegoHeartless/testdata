# Quick Start - Frontend

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev
```

Приложение будет доступно на `http://localhost:5173`

## Конфигурация

Создайте файл `.env` в корне проекта `frontend/`:

```env
VITE_API_URL=http://localhost:3000/api/v1
VITE_API_KEY=test-api-key
```

## Структура приложения

### Страницы

- **/** - Главная страница с генератором профилей
- **/history** - История сгенерированных профилей
- **/profiles/:id** - Детальный просмотр профиля

### Компоненты

- `ProfileGeneratorForm` - Форма настройки генерации
- `ProfilePreview` - Предпросмотр профиля
- `Button` - Переиспользуемая кнопка
- `Card` - Карточка для контента

### Hooks

- `useGenerateProfile` - Генерация нового профиля
- `useProfile` - Получение профиля по ID
- `useProfilesList` - Список профилей
- `useDeleteProfile` - Удаление профиля
- `useExportProfile` - Экспорт профиля

## Использование

1. Откройте главную страницу
2. Настройте параметры генерации:
   - Выберите пол (мужской/женский/случайный)
   - Установите диапазон возраста
   - Выберите регион (опционально)
   - Отметьте нужные документы
3. Нажмите "Сгенерировать профиль"
4. Просмотрите сгенерированный профиль
5. Экспортируйте профиль в JSON или PDF

## Интеграция с бекендом

Frontend автоматически подключается к бекенду через прокси в `vite.config.ts`.

Убедитесь, что бекенд запущен на `http://localhost:3000`

## Сборка для production

```bash
npm run build
```

Собранные файлы будут в папке `dist/`
