## Критерии готовности MVP

### Функциональные требования
- ✅ Генерация полного профиля (ФИО, дата рождения, адрес, паспорт, ИНН, СНИЛС)
- ✅ API с аутентификацией
- ✅ Rate limiting
- ✅ Экспорт JSON
- ✅ Экспорт PDF (базовый)
- ✅ Веб-интерфейс для генерации

### Нефункциональные требования
- ✅ Время ответа API < 500ms
- ✅ Доступность 99%+
- ✅ Документация API
- ✅ Unit тесты (покрытие > 70%)
- ✅ Безопасность (валидация, rate limiting)
